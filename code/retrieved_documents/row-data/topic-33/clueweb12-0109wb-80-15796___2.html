ProductCart: asp shopping cart easy to modify to better fit your needs

What's an asp shopping cart?

In many areas of our Web site we refer to ProductCart as an asp shopping cart. What does "asp" stand for? An asp shopping cart is an ecommerce application written in a popular programming language calledActive Server Pages (ASP). Our shopping cart software is written in ASP.

Why ASP and not ASP.NET

Shopping carts are written in a variety of programming languages such as PHP, ASP.NET, Cold Fusion, Ruby, and more. Classic ASP and ASP.NET are completely different programming language: ASP.NET is not an evolution of Classic ASP, but rather a different programming language. Classic ASP is a scripting language, much more similar to PHP than to ASP.NET.

One of the reasons our customers love ProductCart is that it's so easy to modify. That's because Classic ASP is one of the most accessible programming languages, accessible even for non-developers: Web designers that don't have a programming background can typically learn enough to read and modify at least some ASP source code rather quickly.

Applied to ProductCart, this means that a Web designer working on a Web store powered by our shopping cart software can typically make (at least basic) changes to the software's source code to better meet a client's need. This allows Web designers to have more control on their work and provide a higher level of service to their customers.

Benefits of using Classic ASP

Among the benefits of using an asp shopping cart (shopping cart written in Classic ASP):

  • Your ecommerce store will run on any computer that runs IIS: not just Windows servers, but also Windows XP Professiona, Windows Vista, and Windows 7 desktops on which IIS has been enabled (very easy to do). This means that you can run a ProductCart-powered store locally for designing, developing, and testing purposes, if you need to. For more details on this, seeSetting up a ProductCart-powered store on your desktop.
  • ASP is fast and reliable: it's been around for over a decade and truckloads of Web sites run on it.
  • The ecommerce system's ASP source code is delivered to your when you purchase a shopping cart license.
  • This translates into the ability for you to have any proficient ASP programmer edit the source code to change existing ecommerce features or add new ones. For example, you can hire one of our authorized ProductCart resellers to create a custom ecommerce store for you. Learn more about completeecommerce solutions built with ProductCart.
  • Because ASP is a very popular programming language, you will never have a problem finding skilled programmers that can help you customize, enhance, and maintain your ecommerce store. A great place to start is ourpartners page.
  • Shopping carts written in other programming languanges often do not give you the ability to easily modify the system's features.
