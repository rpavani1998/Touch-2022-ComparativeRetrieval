FAQ


IntensaFIRE for Xbox 360 Frequently Asked Questions



Will this product work with both older and newer Xbox 360 controllers?

This product works with all wireless Microsoft Xbox 360 controllers (CG, matrix, etc.). It does NOT work with wired controllers, only wireless. ones.



What tools & materials are required to install the IntensaFire mod?

Everything you need to install the IntensaFIRE mod is included with the kit, including the special screwdriver and conductive glue. Ordinary household items that you will want to have on hand: Scotch tape, a toothpick, a hair-dryer (for a shorter drying time).



What does the IntensaFIRE mod do?

The IntensaFire modifies Xbox 360 controllers, allowing them to control 5-mode sleeper settings (set up for different games' repetition allowances) with a simple D-Pad or Sync button toggle. It also enables "dual rapid-fire" - usable in all Xbox 360 games.



What games are compatible with this mod?

The IntensaFIRE mod can be used with such gamer favorites as: Left 4 Dead, the Call of Duty series, the Halo series, the Gears of War series, and Street Fighter, as well as any other Xbox 360 game that utilizes a 'rapid-fire' function.

Yes, please view the IntensaFIRE Installation Video for step-by-step instructions on installing and programming the IntensaFIRE kit.



How do I set the programmable mode on my controller?

The IntensaFIRE Installation Video provides step-by-step instructions for programming Mode 5 for the IntensaFIRE.



How can I find out how fast my shooting speed is when programming Mode 5 on my controller?

A Programmable Speed Settings chart is now available.



I've installed the IntensaFIRE, but my right trigger isn't working correctly. Why?

The most common installation error is gluing the three right trigger points together. Each of the three points must be completely separate from the others, if the glue is touching at all, then the mod won't work. Also, be sure that you have attached the sync point correctly for the type of controller you are using.

If you are still having problems, you can try posting a picture of the installed board to theIntensaFIRE Discussion Forum. Our own installation techs and other knowledgeable gamers will offer suggestions there.



After installing the IntensaFIRE, my controller will sometimes fire a random shot, even though I didn't pull the trigger. Does this mean that my mod is defective?

No, it is normal to experience an occasional single random shot from time to time. This is a by-product of the mod programming.



While playing a game with the IntensaFIRE turned on, my controller will sometimes lock up for a moment and stop functioning. Does this mean my mod is defective?

No, it probably means that you are using the wrong mode or speed setting for the game you are playing. If the firing speed is too high, the game will cause your controller to lock up briefly.

The mod itself does not cause this phenomenon, most games have a script or coding that notices rapid fire and reacts by stopping it or locking it up. The way to prevent it is to use the correct pre-set mode for the game you are playing, or else to experiment until you find the fastest speed that does not trigger the game to stop you.
