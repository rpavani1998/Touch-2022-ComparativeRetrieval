Kensington Laptop Desktop USB 2.0

Kensington, Inc.

The Bottom Line

Not all mobile professionals will have a desktop computer at home and rely on using their laptop only. Using your laptop on a desk isn't always the most comfortable or provide you with the best ergonomic position. The Kensington Laptop Desktop USB 2.0 eliminates those problems. It elevates your laptop display to a better viewing position and allows you to use more USB enabled peripherals such as an external mouse, keyboard, external hard drives, printers or scanners.

Pros

  • Increases the functionality of your laptop.
  • Provides a better ergonomic position for your laptop.
  • Has 4 USB 2.0 ports.

Cons

  • The hinge adjustment may be difficult to change with laptop inside.

Description

  • Small footprint - doesn't eat up precious real estate on your desk. Useful if you have a small desk.
  • 4 USB 2.0 ports for external peripherals.
  • Fits laptops 16" and smaller.
  • Dimensions: 15"L x 4.5"H x 12.25"W.
  • Set Up Heights: Highest position: 14.6"L x 5"H x 11.6"W. Lowest position: 11.6"L X 9.5"H X 11.6"W
  • Has a padded insert which provides a secure ensclosure and the curved back enables ventilation.
  • Built-in copyholder keeps documents easy to view and organized.
  • Has a VESA mounting plate which can be used for attaching to monitor arms or wall mounts.
  • Screen height is adjustable, it can be tilted and swiveled.
  • Covered by 1-warranty and free technical support.

Guide Review - Kensington Laptop Desktop USB 2.0

I have an older model Compaq and I've been quite impressed that the Laptop Desktop handles it with no problems. I also used a loaner ultra portable with it. It's been great to have something that works with older, heavier laptop models.

The USB cord placed in the centre back has worked well with both laptops. You can reach either side of the laptop with no problem. There is no hassle connecting the Laptop Desktop to a USB port.

I don’t find the footprint a problem, some laptop desks use the same amount of space, so that hasn't been an issue for me.

The built-in copyholder has certainly been useful and makes it much easier to work from a document, when you don't have to worry about an extra accessory for holding paper.

If you use a wireless mouse with the Laptop Desktop, you've freed up one of the USB ports which can then be used for a scanner, digital camera or other USB enabled accessory. I found it useful being able to scan documents directly to my laptop rather than transferring the images from one computer to the other.

What I've enjoyed the most is bringing the laptop display up to a "normal" level for viewing. This makes it more comfortable to use my laptop regularly at my desk. With two displays, and my laptop going, having everything at the same level is a huge bonus.

Using the Laptop Desktop will provide a great option to purchasing a second desktop for use at-home for mobile professionals and also allow you to work in a better ergnomic environment.
