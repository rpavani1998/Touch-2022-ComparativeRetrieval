Features

Internet Explorer 8 is growing in popularity. We know that many people want to translate text or web pages seamlessly while browsing the Internet.

Now you can install language Accelerators for Internet Explorer 8 and get an instant translation for any web page you are visiting or sections of highlighted text. They work with Firefox too as a plug-in. To enable this functionality in Firefox, a add-on must be installed first which you can downlaod from here.

Simply register and choose the language you want to translate into.

You need to install a different Accelerator for each language pair you need.

Once installed, simply highlight the text you want to translate, click the Accelerator button and see the translated text.
